    How to use this helper

    Find artifacts

    Enter bonus text and select Tag (or press Enter). Results match any active tag, ignoring case. Remove a tag with its × or clear them with Reset tags. A green outline means an artifact matches more than one tag.

    Track what you own

    Select the green ✓ on a result to move it to Owned Artifacts. Select the red × to remove it. Owned artifacts are excluded from search results.

    Choose active bonuses

    In Owned Artifacts, select Active for up to four artifacts; every bonus from those artifacts becomes active. For an artifact that is not Active, select one bonus checkbox to make just that bonus active. The Active bonuses column shows the combined, alphabetized list.

    Save and share

    Your owned and active selections are saved only in this browser’s local storage. Select Copy to create a share string. Send that string to another user; they paste it into Share setup and select Load. Loading replaces their current owned and active setup, then saves it in their browser.
