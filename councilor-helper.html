    DomiNations War Council Helper

    Find councilors by name, type, or War bonus, then select up to seven for your council.

    How to use the War Council helper

    Find councilors

    Search by councilor name, type, or War bonus. Search ignores letter case. Use the type and displayed-rarity filters to narrow the catalog. Matching text is highlighted in the results.

    Build a council

    Select Add to put a councilor in your War Council. You can select up to seven unique councilors. A new selection starts at the rarity currently shown in the catalog. Change each selected councilor’s rarity independently or remove that councilor at any time.

    Review active bonuses

    The Active War bonuses column contains every bonus supplied by the selected councilors at their chosen rarities. The list is alphabetized; repeated bonuses remain visible.

    Save and share

    Your selections and rarities are saved only in this browser’s local storage. Select Copy to put a versioned setup string on the clipboard. Another user can paste that string into the share box and select Load. Loading a valid string replaces the current council; invalid strings leave it unchanged.
