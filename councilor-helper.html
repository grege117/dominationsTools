    How to use the Council helper

    Find councilors

    Choose War or Multiplayer to search that chamber’s bonuses. Search by councilor name, type, or bonus; it ignores letter case. Use the type and displayed-rarity filters to narrow the catalog. Matching text is highlighted in the results.

    Build a council

    Select Add to put a councilor in the selected council. You can select up to seven unique councilors in each mode. A new selection starts at the rarity currently shown in the catalog. Change each selected councilor’s rarity independently or remove that councilor at any time.

    Review active bonuses

    The Active bonuses column contains every bonus supplied by the selected councilors at their chosen rarities. The list is alphabetized; repeated bonuses remain visible.

    Save and share

    War and Multiplayer selections and rarities are saved separately in this browser’s local storage. Select Copy to put a versioned setup string on the clipboard, or Copy Link for a read-only share URL. Opening a share URL does not change the viewer’s saved setup. Another user can paste a setup string into the share box and select Load; loading switches to the shared mode and replaces that council only. Invalid strings leave everything unchanged.
